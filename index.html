Apa kepanjangan dari XAMPP?

X (cross platform), A (Apache), M (MySQL/MariaDB), P (PHP), dan P (Perl)

Bagaimana cara untuk mengakses XAMPP Panel?

download apachefriends.org. lalu install di komputer masing masing

Module apa yang harus diaktifkan untuk menjalankan MYSQL Server?

Apache dan MySQL

Bagaimana cara menjalankan PHPMyAdmin di browser?

pertama jalankan XAMPP pilih modul apache lalu buka localhost. Setelah terbuka lalu klik pojok kanan atas yang bertuliskan PHPMyAdmin

Bagaimana menjalankan MYSQL Client?

nyalakan xampp pilih modul apache & mysql lalu Buka CMD line lalu ketik c:\xampp\mysql\bin\mysql -u root setelah dijalankan memunculkan Welcome dari MariaDB berarti sudah terjalankan
